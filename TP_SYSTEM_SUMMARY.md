## 五、EVT止盈特性

### 5.1 与传统ATR止盈的区别

| 特性 | ATR固定止盈 | EVT极值止盈 |
|------|------------|-------------|
| 依据 | 历史波动率 | 历史极值分布 |
| 适应性 | 线性调整 | 非线性（厚尾） |
| 数据利用 | 全部数据 | 极值数据（95%+） |
| 数学基础 | 标准差 | GPD分布 |
| 对ETH适配 | 一般 | 优秀（厚尾） |

### 5.2 EVT参数说明

```python
# 关键参数
ξ (shape): 形状参数，ETH通常在0.2-0.4之间（厚尾）
β (scale): 尺度参数，极值分散程度
u (threshold): 阈值，通常取95%分位数

# 计算公式
止盈收益率 = (阈值 + GPD_分位数) × 安全折扣 × 环境调整
```
