```python

if is_counter_trend and ml_confidence < 0.82:
    return HOLD  # 观望，不交易
```

**预期效果**：趋势市胜率从5-12%提升至40%+

### 2.2 各市场环境策略详解

#### 【震荡市】SIDEWAYS
- **入场**：价格触及布林带下轨(做多)或上轨(做空)
- **过滤**：
  - 布林带宽度≥0.03
  - ML置信度≥0.70
  - RSI6<30或成交量>1.2倍
- **止盈止损**：止损1.35倍ATR，止盈至中轨
- **胜预期率**：60%+

#### 【趋势上涨】TRENDING_UP
- **入场**：ML预测涨 + 顺势
- **过滤**：
  - 逆势交易需置信度≥0.82
  - 资金费率不过高
- **止盈止损**：止损1.5倍ATR，止盈8倍ATR(让利润奔跑)

#### 【趋势下跌】TRENDING_DOWN
- **入场**：ML预测跌 + 顺势
- **过滤**：同上

#### 【突破市】BREAKOUT
- **识别**：价格创20根新高 + ADX>40 + 成交量>1.5倍
- **策略**：追涨，收紧止损(1.5倍ATR)，大止盈(10倍ATR)
- **风险**：假突破风险高，必须放量确认

#### 【暴跌市】BREAKDOWN
- **识别**：5分钟跌1.5%+放量+价格<布林带下轨
- **策略1**：RSI<20极度超卖时，轻仓试多(抓反弹)
- **策略2**：ML看跌时追空
- **风险**：恐慌性抛售可能持续，需严格止损

#### 【暴涨市】PUMP
- **识别**：5分钟涨1.5%+放量+价格>布林带上轨
- **策略1**：RSI>80极度超买时，轻仓试空(抓回调)
- **策略2**：ML看涨时追多
- **风险**：FOMO情绪可能延续，需谨慎

#### 【高波动市】HIGH_VOL
- **识别**：ATR>1.5%但ADX<25(无明显趋势)
- **策略**：提高置信度门槛至0.75，减仓操作，收紧止盈止损
- **风险**：假信号多，容易双向止损

#### 【低波动市】LOW_VOL
- **识别**：ATR<0.3%+成交量<0.7倍+布林带宽度<0.02
- **策略**：禁止交易
- **原因**：流动性枯竭，点差大，难盈利

#### 【反转市】REVERSAL
- **识别**：
  - 上涨趋势+价格高位+成交量萎缩(顶背离)
  - 下跌趋势+价格低位+成交量萎缩(底背离)
- **策略**：ML置信度≥0.80时，小仓位(0.7倍)逆势试探
- **风险**：判断错误代价高，需极高置信度

#### 【盘整市】CONSOLIDATION
- **识别**：布林带收窄且持续20根K线以上
- **策略**：在布林带极端位置(0.15/0.85)等突破
- **预期**：即将选择方向，提前布局
