## 三、核心策略详解

### 3.1 趋势市顺势策略（修复后）

```python
# 顺势交易
趋势上涨 + ML看涨 → BUY
趋势下跌 + ML看跌 → SELL

# 逆势交易（抓回调，需极高置信度）
趋势上涨 + ML看跌 + 置信度<0.82 → HOLD (拒绝)
趋势上涨 + ML看跌 + 置信度>=0.82 → SELL (允许)
```

### 3.2 ETH插针暴跌策略

```python
识别: 3分钟跌>1% + 价格≈20分钟低点 + 放量>1.3

策略1: 抄底（风险极高）
- 条件: RSI<25 + RSI6<20 + ML置信度>0.72
- 止损: 0.8倍ATR（极紧）
- 止盈: 4倍ATR（抓反弹）

策略2: 追空（趋势延续）
- 条件: ML看跌 + RSI>30（不是极度超卖）
- 止损: 1.5倍ATR
- 止盈: 5倍ATR

默认: 观望（最安全）
```

### 3.3 ETH爆拉策略

```python
识别: 3分钟涨>1% + 价格≈20分钟高点 + 放量>1.3

策略1: 摸顶做空（风险极高）
- 条件: RSI>80 + RSI6>85 + 缩量 + ML看跌
- 止损: 1.0倍ATR（防止继续爆拉）
- 止盈: 3倍ATR

策略2: 顺势追多
- 条件: ML看涨 + RSI<75 + 放量 + 资金费率优惠
- 止损: 1.0倍ATR

默认: 观望（避免FOMO）
```

### 3.4 突破追势策略

```python
识别: 价格>20分钟高点 + ADX>35 + 放量>1.3 + 均线多头排列

条件: 
- ML看涨 + 置信度>0.68 + RSI<75（防止追顶）
- 资金费率<0.001（做多成本可控）

止盈止损:
- 止损: 1.2倍ATR（ETH突破后波动大）
- 止盈: 12倍ATR（ETH趋势惯性大）
```

### 3.5 高波动洗盘策略

```python
识别: ATR>1.2% + ADX<25（大波动无方向）

ETH特点: 多空双杀，上下插针

策略:
- 门槛: ML置信度>0.78（极高）
- 位置: 只在布林带极端位置交易
  - 做多: bb_position<0.4
  - 做空: bb_position>0.6
- 仓位: 0.85倍（减仓）
- 止损: 0.8倍ATR（极紧）
- 止盈: 2.5倍ATR（不贪）
```

### 3.6 盘整待破策略

```python
识别: 布林带收窄持续>15分钟

ETH特点: 盘整后往往有大行情

策略:
- 下轨埋伏(<0.1): 预期向上突破
- 上轨埋伏(>0.9): 预期向下突破
- 止损: 1.0倍ATR
- 止盈: 8倍ATR（突破空间大）
```
