# ETHUSDT 市场辅助数据集成说明

## 一、新增模块

### binance_data_feed.py
提供币安市场辅助数据，包括：
- 多空比 (Long/Short Ratio)
- 爆仓数据 (Liquidation Data)
- 持仓量 (Open Interest)
- 综合市场情绪

**重要原则**: 这些数据仅作为辅助参考，不主导交易决策！

---

## 二、数据使用方式

### 2.1 轻量级调整策略

```python
# 市场数据仅做轻微调整（±5%以内）

# 1. 多空比极端时
if 多空比 > 4.0 且 当前信号是做多:
    置信度 -= 5%   # 轻微降低，不反转信号
    理由 += "[多空比极端多头，-5%]"

# 2. 爆仓数据极端时
if 多头爆仓激增 且 当前信号是做空:
    置信度 -= 3%   # 不再追空
    理由 += "[多头爆仓多，-3%]"
```

### 2.2 不主导决策的体现

| 场景 | 如果主导决策 | 实际实现（辅助） |
|------|-------------|----------------|
| 多空比>4 | 做空 | 做多置信度-5%，仍为做多 |
| 多头爆仓激增 | 抄底做多 | 做空置信度-3%，谨慎追空 |
| 多空比<0.25 | 做多 | 做空置信度-5%，仍为空 |

**核心原则**: 主策略（ML+技术）决定方向和是否交易，市场数据仅微调置信度。

## 六、风险控制

### 6.1 防止过度依赖

```python
# 调整限制
max_adjustment = 0.05  # 最大调整5%

# 即使市场数据极端，也不反转信号
if 主信号是做多 and 多空比极端:
    只做微调，不改为做空
```

### 6.2 API故障保护

```python
# 缓存机制
if API调用失败:
    使用缓存数据（5分钟内有效）
if 缓存也过期:
    忽略市场数据，主策略继续运行
```
