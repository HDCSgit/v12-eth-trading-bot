# V2.5-Hybrid 混合策略系统

## 🎯 策略概述

V2.5-Hybrid 是在 V2 双信号系统基础上的**轻度激进优化版**，融合了：
- ✅ V2 双信号叠加（高胜率过滤）
- ✅ 轻量双向网格（震荡市盈利）
- ✅ 斐波那契动态补仓（成本优化）
- ✅ 三层智能止盈（风险控制）

**目标**：年化收益 110%-135%，最大回撤 ≤14%

## 🚀 快速开始

### 1. 配置文件 (config.py)

```python
CONFIG = {
    "SYMBOLS": ["ETHUSDT"],
    "INTERVAL": "1m",
    "LEVERAGE": 5,           # 默认5x，可选10x
    "MODE": "LIVE",          # PAPER/LIVE

    "MAX_DD_LIMIT": 0.15,
```

## 📈 交易流程示例

### 场景：ETH 从 $2130 下跌后的开多流程

```
Step 1: 信号触发
  RSI: 34 (超卖) + MACD企稳 + BB下轨 = 2信号
  置信度: 0.60 → 触发 BUY
  开仓: $2130, 数量: 0.007 ETH (风险$3)

Step 2: 网格初始化
  ATR: $8.5, 网格间距: $8.5×1.4 = $11.9
  Grid Level 1: $2141.9 (止盈50%仓位)
  Grid Level 2: $2153.8 (止盈25%仓位)
  Grid Level 3: $2165.7 (止盈12.5%仓位)

Step 3: 价格下跌 → 触发补仓
  价格跌至 $2115 (下跌$15, >1.2×ATR×1.0)
  DCA #1: 补仓 0.007×1.0 = 0.007 ETH @ $2115
  新均价: ($2130×0.007 + $2115×0.007) / 0.014 = $2122.5

Step 4: 价格继续下跌
  价格跌至 $2095 (下跌$20, >1.2×ATR×1.618)
  DCA #2: 补仓 0.007×1.618 = 0.011 ETH @ $2095
  新均价: ($2122.5×0.014 + $2095×0.011) / 0.025 = $2110.4

Step 5: 价格反弹 → 触发止盈
  价格涨至 $2140 (触及 Grid Level 1)
  平仓 50%仓位 (0.0125 ETH) @ $2140
  盈利: ($2140 - $2110.4) × 0.0125 = $0.37

Step 6: 继续上涨 → 追踪止盈
  价格涨至 $2150, 回撤至 $2138
  触发追踪止盈 (从高点回撤3%)
  平仓剩余全部仓位
  总盈利: ~$2.5 (本金的4.3%)
```

## 🛡️ 风险控制

### 1. 保证金安全垫
- 始终保留 35% 以上保证金
- 防止强平和爆仓

### 2. 瀑布防护
- 5分钟跌幅 > 4% → 暂停所有补仓
- 防止连续补仓导致风险失控

### 3. 连续止损保护
- 连续 2 笔止损 → 当日暂停新仓
- 避免震荡市反复亏损

### 4. 逐仓独立风控
- 每笔订单独立计算止损
- 互不影响

## 📊 数据库结构

V2.5 使用独立的数据库 `elite_trades_v2_5.db`：

```sql
-- 交易记录表
trades (id, timestamp, symbol, action, qty, price, pnl, pnl_pct, 
        reason, order_id, confidence, trade_type, dca_index, fib_multiplier)

-- 网格层级表
grid_levels (id, timestamp, symbol, level_index, target_price, qty, 
             side, filled, filled_time, pnl)

-- 仓位组表
position_groups (id, symbol, side, entry_price, total_qty, dca_count, 
                 avg_price, max_profit, is_active, closed_time)
```

## 🔍 关键参数说明

| 参数 | 值 | 说明 |
|------|-----|------|
| `grid_atr_multiplier` | 1.4 | 网格间距系数（轻度激进） |
| `dca_fib_sequence` | [1.0, 1.618, 2.618] | 斐波那契补仓系数 |
| `dca_accelerator` | 1.2 | 第3次补仓加速系数 |
| `max_dca_count` | 3 | 最大补仓次数 |
| `grid_profit_target` | 0.008 | 单格止盈目标 0.8% |
| `trailing_start` | 0.06 | 追踪止盈启动 6% |
| `trailing_stop` | 0.03 | 追踪止盈回撤 3% |
| `group_take_profit` | 0.08 | 整组止盈 8% |
