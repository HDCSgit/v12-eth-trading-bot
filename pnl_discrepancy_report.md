## 问题根源分析

### 问题1：手续费率设置错误
```python
# config.py 当前设置
"FEE_RATE": 0.0004  # 0.04% Taker

# 币安实际费率
Taker: 0.05%  # 系统低估了25%
Maker: 0.02%
```

**影响**：
- 系统假设每轮交易费用：0.08%
- 实际每轮交易费用：0.10%
- 每轮多损耗：0.02%

### 问题2：PnL计算方式错误（推测）

系统可能按以下方式计算：
```
PnL% = (出场价 - 入场价) / 入场价 * 杠杆
```

但币安实际PnL计算：
```
Realized PnL = (出场价 - 入场价) * 数量 - 开仓手续费 - 平仓手续费
```

**关键差异**：
- 系统可能按名义价值计算收益率（包含杠杆）
- 币安按实际保证金计算
- 例如：5x杠杆下，1%价格变动 = 系统显示5%收益，实际币安显示1%

### 问题3：所有成交都是Taker

- 系统使用市价单，全部为Taker
- Taker费率0.05%比Maker的0.02%高2.5倍
- 如果部分成交能用Maker，可节省60%手续费
